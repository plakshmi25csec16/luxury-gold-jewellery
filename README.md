# Aurum&Co — Luxury Gold E-Commerce Home Page

Premium responsive gold-themed e-commerce landing page built with **React 18 + Vite + Tailwind CSS v4**.

## Sections
Navbar · Hero · Categories · Products · Offer Banner · Reviews · Brands · Footer

## Run

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Structure

```
src/
  assets/              # product & hero images
  components/site/     # Navbar, Hero, Categories, Products, OfferBanner, Reviews, Brands, Footer
  App.tsx
  main.tsx
  styles.css           # Tailwind v4 + gold design tokens & animations
```
